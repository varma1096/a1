Smart Calculator Help

Smart Calculator is a powerful tool to do mathmatic analysis. It supports: 1. math expression recognition (need back camera); 2. solve mathematic equation(s); 3. complex number; 4. array (matrix); 5. (higher level) integration; 6. 2D, polar and 3D chart plotting and 7. string besides the basic calculations provided by most calculator apps. The calculator is actually derived from another powerful calculator app named Scientific Calculator Plus but it disables its programming capability and adds math expression recognition. As such it is more easy-to-use. User can input using built-in input pad, system soft keyboard or camera, and Smart Calculator employs a powerful programmable math engine called MFP to analyse mathematical expressions. As such it can satisfy most of user's requirements.
